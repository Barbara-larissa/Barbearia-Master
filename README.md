# ✂️ Barbershop Landing Page

Uma landing page moderna, elegante e totalmente responsiva desenvolvida para barbearias que buscam uma presença online profissional. O foco deste projeto foi criar uma interface de usuário (UI) atraente com uma experiência de navegação fluida.

---

## 🚀 Tecnologias Utilizadas

Este projeto demonstra o uso de tecnologias modernas de Front-End:

* **HTML5** - Estruturação semântica para melhor SEO.
* **CSS3** - Estilização personalizada com foco em design moderno.
* **JavaScript** - Interações dinâmicas e efeitos de rolagem.
* **Google Fonts** - Tipografia selecionada para um ar sofisticado.
* **Lucide Icons / Font Awesome** - Ícones leves e elegantes.

---

## ✨ Destaques do Projeto

* **Design Responsivo:** Adaptado perfeitamente para dispositivos móveis, tablets e desktops.
* **UI/UX Premium:** Cores escuras e contrastantes que remetem ao ambiente clássico de uma barbearia.
* **Seções Estratégicas:**
    * **Home:** Banner de impacto com chamada para ação (CTA).
    * **Serviços:** Lista de cortes e tratamentos com preços.
    * **Sobre:** A história e a expertise dos barbeiros.
    * **Contatos/Localização:** Integração visual para facilitar o agendamento.
